# IW-L1.2-MiniProject1-JavaScriptEssentials

## How can I store and manipulate data to later be used in various ways throughout my website?

This vanilla javascript lab is a series of challenges for you to get familiar with javascript essential topics from variable arithmetics to functions with conditional statements. 

You will be required to research various methods, the Pythagorean theorem formula, and  recursive functions to be completed.
